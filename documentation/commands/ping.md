# /Ping

## Usage of /ping

{% stepper %}
{% step %}
The ping command doesn't has any input value, just type in the base command.

| command | value   |
| ------- | ------- |
| /ping   | \<none> |
{% endstep %}

{% step %}
The bot will return the latency in milliseconds (ms)

> Ping : 2ms
{% endstep %}
{% endstepper %}
